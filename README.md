⌨️ Typing Test App

An interactive Typing Speed Test Application built using HTML, CSS, and JavaScript.
This app helps users improve their typing speed and accuracy by providing real-time feedback such as WPM, CPM, mistakes, and timer.

🚀 Features
Real-time typing speed calculation (WPM & CPM)
Tracks typing mistakes
60-second countdown timer
Dynamic paragraph generation
Sound effects for correct and incorrect typing
Restart functionality
Clean and responsive UI

🛠️ Technologies Used
HTML
CSS
JavaScript (DOM Manipulation)

💡 How It Works
A random paragraph is displayed using JavaScript
User starts typing in the input field
Timer starts automatically on first keystroke
Each character is validated:
Correct input → marked green + sound (optional)
Incorrect input → marked red + mistake count increases
WPM and CPM are calculated in real-time
User can restart the test anytime

📊 Metrics Explained
WPM (Words Per Minute): Typing speed calculation
CPM (Characters Per Minute): Total correct characters typed
Mistakes: Number of incorrect inputs
